# cosmic-thread
A mystery-solving game set in the far-away Arsemax Cluster, on the planet Xaltra.  In 2025, you play as the universal agent tasked with unraveling a murder mystery. The game features a single investigation board scene where players click on items to uncover clues and solve the case.

## Development
This project was developed for Interactive Narrative and PRogramming for Games classes at NYU, but it quickly became a passion projects with my friend Sahra Qadi, who helped with writing the story bible. Cosmic Thread is currently in progress—this demo version is made in GameMaker, but the official release will be on Unity.  

I always wanted to make a murder-mystery game, but wasn't fond of drawing regular people, so I drew aliens! It started with the game title then got inspo for the Xaltarians' concept art :3

## Usage
The game is designed for single-player gameplay.  Players interact with the investigation board by clicking on items to gather clues and advance the narrative. The goal is to piece together the clues and solve the mystery.

## Installation
The game is available as an executable file from itch.io. Install and run the executable to begin playing.
